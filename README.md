# Nestle Nigeria: 3-Year Financial Analysis & Rolling Forecast (2024–2026)

## 📌 Project Overview
This project delivers a rigorous financial evaluation and forward-looking forecast for Nestle Nigeria PLC. It combines historical data analysis with predictive financial modeling to assess the company's fiscal health, operational efficiency, and growth trajectory through 2026.

## 📊 Key Deliverables
*   **3-Year Historical Analysis:** Deep-dive evaluation of income statements, balance sheets, and key corporate financial metrics.
*   **Rolling Financial Forecast (2026):** Pro-forma predictive financial modeling based on historical performance, market trends, and macroeconomic indicators.
*   **Financial Ratio Modeling:** Detailed calculation and visualization of liquidity, profitability, and leverage ratios.

## 🛠️ Tools & Technologies Used
*   **Microsoft Excel:** Advanced financial modeling, forecasting, and data visualization.
*   **Financial Planning & Analysis (FP&A) Frameworks:** Scenario analysis and rolling forecasting methodologies.

## 📂 Included Files
*   `NESTLE FP&A 2.xlsx` - Full interactive financial model and data calculations.
*   `NESTLE PDF FILE.pdf` - Completed executive presentation and summary report.
